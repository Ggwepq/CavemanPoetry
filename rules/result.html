Rules for Poetry for Neanderthals

1. Players must explain words using only single-syllable words.

2. Each card has words worth 1 point and 3 points.

3. Successfully explaining a 1-point word earns 1 point.

4. Successfully explaining a 3-point word earns 3 points.

5. Skipping a card results in a 1-point deduction.

6. The game ends when the timer reaches zero.
